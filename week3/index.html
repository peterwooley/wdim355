Events

  • jQuery has extensive support for events beyond what we've played with, including the ability to bind multiple events, unbind events, and trigger custom events.
  • We can utilize the bind() and trigger() methods for these advanced uses.
  • You can use the optional 'data' parameter of the bind method to pass along data into the event.data scope.
  • All jQuery objects can call any custom events.
