Flickr Gallery

  • Create a JavaScript Web Application that allows a user to search for images on Flickr by tag name and display them in a gallery.
  • The UI should allow a user to enter a tag (or multiple tags using a comma-separated list) into an input field.
  • The App should function similarly:
    • The user types in a Flickr tag they're looking for.
    • The images are presented to them in a gallery-format (How the gallery works is up to you!).
    • At any time, a user should be able to change their tag list and get an updated gallery.
