My Debugging Process

Problem: My code isn't running!

  1. Check for errors in Firefox's Error Console (Tools » Error Console)
    • Clear the console.
    • Refresh the page with the broken JavaScript.
    • Observe the code not working.
    • Switch back to the Error Console and check for errors.
    • If there's an error listed, find it, fix it, check if that fixed it.
  2. If there are no errors, I pop open Firebug's Script Panel.
    • Enable the Script Panel if it's not already enabled.
    • Set a breakpoint on a line I would expect to execute.
    • Refresh the page or cause the event that should trigger the code.
    • If the breakpoint is triggered, I step through the code until it breaks, adding new breakpoints if necessary.
    • If the breakpoint is not triggered, I add a new breakpoint earlier and earlier in the execution cycle, even back to the first line in the first <script />.
  3. Failing the first two (which is really, really rare), I check to ensure that JavaScript is enabled in my browser, restart the browser, and pray to the JavaScript gods.
  4. If all of those fail, I add an alert() to the very first line of my script and if it fires, I slowly move it down, line by line, until I find the broken line.
