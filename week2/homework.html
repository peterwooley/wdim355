Assignmeent 1, due Week 3.

When finished, please upload this assignment to the Drop Box on eCompanion.

  • Create a JavaScript Widget that allows a user to customize the text color and background color of a bit of content.
  • The User should have two input boxes where they can enter in hex values.
  • When the user enters a CSS color value (#fff, red, etc.) into one of the inputs, the associated color should automagically change.
  • Important: You must be able to create multiple copies of the Widget by adding a single line of JavaScript for each copy.
