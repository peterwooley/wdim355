Homework: Assignment 3, due Week 4.

Please turn in your files in a .zip file, using the following naming convention: lastnameFirstname_assignment3.zip

  • Create a JavaScript Web Application that accepts a form submission of a textarea and appends the content to a reverse chronological sorted list of messages (Yeah, like Twitter).
  • The UI should have, at least, a title, textarea and submit button, and reverse chronological list of previous messages.
  • The App should function similarly:
    • The user types in their message, hits submit, and the message is added to the top of the messages list.
    • When a message is submitted the form should be emptied.
    • At any time, a user should be able to delete an old message.
  • Tip: Create a single object that binds events to the form submission utlizing jQuery, other frameworks, or vanilla DOM access.
  • Tip: Utilize a Message object that is stored in the main App object, which allows for the deletion of the message from the UI.

Extra Extra Credit (30% Bonus):

  • Implement Ajax functionality to store previous messages, so users can login and see their old messages after a page refresh.
