Homework: Midterm Project, due Week 5

jQuery Plugin

  • Create a jQuery Plugin that performs one of these tasks:
    • Finds all urls in the provided jQuery collection and turns them into links.
    • Translates all of the text in the provided jQuery collection into Leet.
    • Causes all of the elements provided in the jQuery collection to animate toward the user's cursor.
    • Adds a unique Lolcat image to each element in the jQuery collection.
  • The plugin should be chainable. (e.g., $("div").linkify().css("background", "fuschia"); should work.)
  • The plugin should be stored in a separate .js file with this convention: jquery.[pluginName].js.
  • The plugin should use the suggested code template: 
    (function($){
  $.fn.myPlugin = function() {
    // Do your awesome plugin stuff here
  };
})(jQuery);
