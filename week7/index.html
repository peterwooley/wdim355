altJS

JavaScript is the lingua franca of the web, but that doesn't mean you have to write JavaScript when you're programming web sites or web apps.

A lot of languages exist that compile down to JavaScript. By doing so, these languages can look and feel different than JavaScript, but still run in all web browsers. A few of them are:

  • CoffeeScript
  • Microsoft's TypeScript
  • StratifiedJS
  • ClojureScript
  • Google's Dart

Introduction to CoffeeScript

"CoffeeScript is an attempt to expose the good parts of JavaScript in a simple way."

# CoffeeScript
square = (x) -> x * x
// JavaScript
var square;
square = function(x) {
  return x * x;
};
