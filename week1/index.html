What Object-oriented is and why to use it

  • It's using custom objects to handle their own properties and methods and interacting with each other in predictable ways.
  • Remember:
    • Everything is an Object in JavaScript
    • JavaScript is an Object-Oriented Language

Ways to create custom objects

  • Primitive 
    var MyObject = {
	title: 'A Custom Object',
	markupTitle: function() {
		return '<h1>' + this.title + '</h1>';
	}
}	
var mine = MyObject;
alert(mine.markupTitle());
  • Object Constructor 
    function MyObject(title_arg) {
	this.title = title_arg ? title_arg : 'A Custom Object';
	
	this.markupTitle = function() {
		return '<h1>' + this.title + '</h1>';
	}
}
var mine = new MyObject('A totally awesome custom object!');
alert(mine.markupTitle());

Encapsulation: Public and Private

  • Utilizing an Object-oriented structure, you can limit how objects can interface with other objects.
  • These items do not exist as keywords, as in some languages, but as patterns.
  • Public 
    function MyObject(title_arg) {
	this.title = title_arg; // Available from anywhere
}
  • Private 
    function MyObject(title_arg) {
	var title = title_arg; // Only available inside the object
	
	this.get_title = function() {
		return title;
	}
}

Inheritance: It's prototyping time!

  • JavaScript objects may inherit properties and methods from each other using their prototype property.
  • JavaScript is actually considered a Prototyped-based programming as there are no Classes to inherit each other.
  • Using prototyping, you can create children objects that have access to the properties and methods of their parent.

Inheritance: It's easy possible

function Animal(name) {
    this.type = "animal";
    this.getName = function() { return this.name };
}

function Felinae(name) { // See http://en.wikipedia.org/wiki/Felinae
    this.name = name;
	this.pur = function() { return this.name + " purs." };
}
Felinae.prototype = new Animal();

function Cat(name) {
    this.name = name;
	this.scratchCouch = function() { return this.name + " scratches couch to death." };
}
Cat.prototype = new Felinae();

var zoe = new Cat("Zoe");
console.log(zoe.getName()); // logs "Zoe"
console.log(zoe.pur()); // logs "Zoe purs."
console.log(zoe.scratchCouch()); // logs "Zoe scratches couch to death."
