Diving; due Week 2.

  • Create a custom object that accepts an argument of an image url in its constructor and stores that url in a property named source.
  • Give it a public method named draw that ues jQuery to create a new <img> element with the source property as its src attribute and append the <img> element to the <body>. (See the jQuery Documentation on creating elements with $(), the attr() method, and the appendTo() method.)
  • Create two instances of the custom object with different valid image references and call the draw method on both.
